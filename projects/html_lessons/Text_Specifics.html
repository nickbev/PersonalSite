Other Ways to Manipulate Text

You can use strong to show importance of a word or phrase, which typically is seen as bolded by default.

Similarly, you can use em to show emphasis which is default seen as italics.

This is a use of block quote for longer quotes that come from another website, and are seen as indented.

A more direct way to use quotations is with q, which places a pair of quotation marks between the calls.

By using the abbr element, you can simplify acronyms with information about their full name. (Example = EHS).

When citing a source, you use cite, which usually appears as italics (Example = The Adventures of Huckleberry Finn).
