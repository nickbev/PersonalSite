Iframes

Like a little window pasted onto your webpage, frames can be taken form other pages and seen on yours, for example Google Maps. You can also set its width and height. You can also set whether you can scroll on the site or have a border with scrolling and frameborder attributes.

Page Information

Using the meta element allows information about the webpage. Although not visible to the user, it tells engines about the page and usually has name and content attributes. Some usable variables include description, keywords, robots, author, pragma, and expires.
