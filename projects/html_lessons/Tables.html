Tables

Structure

Within a table, you can use tr to represent each row and inside, a td will represent your data in a row.

1 2 3
4 5 6
7 8 9

Headings

If you use th instead of td, you can indicate a heading of a column or row, and with the scope attribute, you can specify with the row or column value.

Column 1 Column 2
Row 1 A B
Row 2 C D

Spanned Columns and Rows

By using td with the attribute rowspan or colspan with a value, you can make a value span for a number of spaces horizontally or vertically.

Column 1 Column 2 Column 3
Row 1 A B
Row 2 C Shifted Spot

Organization for Longer Tables

With a thead, tbody, and tfoot, you can create an organized table with a result.
