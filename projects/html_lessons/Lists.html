Lists

Ordered (Numbered) List

Using ol, you can group a number of things together to create a list of steps or rankings or whatever.

  1. Step 1
  2. Step 2
  3. Step 3
  4. Step 4
  5. Step 5

Unordered List

Using ul, you can group a number of things together to create a bulleted list.

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

Definition List

Using dl, you can group a number of things together which each use dt (term) and dd (definition) to complete a list of vocabulary words or such.

Item 1
Definition 1
Item 2
Definition 2
Item 3
Definition 3
Item 4
Definition 4
Item 5
Definition 5

Nested List

Similar to a nested loop in Java, you can create a list inside another to create a sub-list, seen as another form of bullet points. Do not end the li until after the sub-list.

  • Item 1
  • Item 2
    • Sub-item 1
    • Sub-item 2
  • Item 3
