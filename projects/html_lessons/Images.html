Adding An Image

You can add the image using img with information such as the source (src), description of not visible (alt), and title when hovered over (title).

Adding An Image with Adjusted Dimensions

You can set the width and height of the image as well.

Where you place an image

If you place your image..

  1. before a paragraph, the image appears in a line above.
  2. in the beginning of a paragraph, the image is on the same line in front.
  3. in the middle of a paragraph, the image is fitted inside.

Types of Images

Some image types include..

  • JPEG, which is best for photographs with various colors throughout.
  • GIF, which is best for images with few colors or designed on a computer.
  • PNG, which is mostly used for pictures with various levels of transparency.
