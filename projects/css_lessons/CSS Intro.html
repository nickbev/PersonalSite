CSS Intro

Understanding CSS

Imagine every HTML element surrounded by an invisible box.

CSS creates rules that control how each box (and their contents) is presented.

You can make box changes (width, height, borders, colors, position) and text ((typeface, size, color, upper + lowercase) and other specific examples.

Structure and Effects of CSS

CSS rules have two parts: a selector and a declaration.

Following the single or multiple (with commas between) elements, you use brackets where between you set properties and their respective values.

External CSS

In the head element where the title goes, you make a link element to the CSS document.

You also use the href attribute for the path, the type attribute for document (text/css) and rel for its relationship (stylesheet).

Internal CSS

You can also create CSS within an HTML document using the style element. You must have the attribute type with value text/css to indicate it is CSS.

This is only more efficient when applying a style to one page. If you have multiple pages with the same formatting, you should have a separate css document to reference.

CSS Selectors

Using various commands you can pick particular elements within an HTML document for particular CSS rules to be applied.
